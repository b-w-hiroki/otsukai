# バナー画像生成用プロンプト集

ChatGPT (Image / DALL·E / Sora) や他の画像生成AIで「おうちのおつかい」のバナーを作るときに使えるプロンプト集。
ログイン画面に載せた機能説明と整合する内容になっている。

---

## ブランド設定(全プロンプト共通)

- **アプリ名**: おうちのおつかい(Ouchi no Otsukai)
- **コンセプト**: 家族でおつかい・在庫・ミッションを共有するスマホアプリ
- **カラーパレット**:
  - メイン: 紫グラデーション `#7c3aed → #5b5ef4 → #818cf8`
  - サブ: クリームベージュ `#faf9f7`(背景)、ホワイト `#ffffff`(カード)
  - アクセント: インディゴ `#6366f1`、ソフトラベンダー `#eeecfb`
- **モチーフ**: 🧺(かご)、🛒(カート)、📱(スマホ)、🥚🥛🥕(食材)、🎯(ミッション)
- **トーン**: あたたかい、やさしい、現代的、日本の家庭、ほっとする
- **サイズ**: 横長バナー(`1200×630` PR/OGP用 / `1920×600` ヒーロー用 / `1080×1080` SNS正方形)

## 使い方Tips

- **文字は別途乗せる**: 画像生成は日本語タイポグラフィが苦手なので、画像生成後にFigma / Canva / Photoshopで「おうちのおつかい」タイトルを乗せると綺麗に仕上がる。
- **アスペクト比は明示**: ChatGPT Imageは指定するとそれに従う。指定がないと正方形気味になりがち。
- **複数生成して合成**: 例えばプロンプトAでメイン背景を生成 → 別画像でロゴを生成 → 合成、の方が制御しやすい。
- **NSFW/不適切要素回避**: 子どもが登場するシーンでは「cute simple cartoon child, no realistic features」のように指定すると安全寄りに振れる。
- **OGP/Twitterカード**: 1200×630を採用、テキスト領域は上または左を確保。
- **ヒーロー用**: 1920×600 や 2400×800 に変更し、横長で広がりを出す。
